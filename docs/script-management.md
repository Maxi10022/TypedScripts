# Script Management

For TypedScripts to pick up a shell script a few conditions must be met:

* ***Build Action*** must set to ***Additional Files***.
* The script must end with the ***.sh*** extension

Make sure your scripts meet those requirements, otherwise there wont be happening any codegen for them.  
Refer to the [argument definition docs](/argument-definition) to learn how to configure arguments for your scripts.
