# Supported Argument Types

The following argument types are supported to be configurable via the `@param` comments in shell scripts. They're grouped for easier scanning.

**Text**
- `string` — text
- `char` — single character

**Boolean**
- `bool`

**Signed integers**
- `sbyte`, `short`, `int`, `long`

**Unsigned integers**
- `uint`, `ushort`, `ulong`

**Floating-point & decimal**
- `float`, `double`, `decimal`
